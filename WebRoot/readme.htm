《地球的血液》主题网站

技术说明文档

      

 

1、建站目的及功能定位

1.1 建站目的

我们的网站响应主题“守护青山绿水蓝天”，我们选取了其中一个“绿水”作为主题，以“地球的血液”为题来突出。众做周知，随着社会的发展，大力开展工业的同时，我们的水资源没有得到保护，一天一天恶化。所以，守护绿水当务之急。我们的网页正是要达到让大家都一同来守护我们的生命之源，地球的血液――绿水。 在我们的网站中，详细的介绍了水资源的污染状况，以及近年来志愿者，环卫工人，国际国家的各种组织所开展的治水活动，以及他们所搞的环保工作，大到国家，小到个人，每个人都在为还地球一片绿水做努力。给人正能量的同时也能号召大家一起为治水而努力。

1.2 功能定位

根据网站资源和功能，定位网站为宣传性、学习性的展示平台。

2、网站内容规划

2.1 网站名称

我们的网站名称：《地球的血液》

2.2 网站主题

展示地球特别是中国水资源面临的问题，国家政策以及志愿者等进行的治水活动，号召全民治水护水，为碧绿江湖河水贡献自己的一份力。

2.3 网站语言

 简体中文/English

2.4 网站风格

我们的网站属于环保型的网站，侧重点在水，采用了蓝色做为网站的总体色，使用浅灰色作为背景色，偶尔穿插多姿多彩的图片和白色背景使得网站的层次非常分明，视觉上感觉很舒适。

2.5 网站内容

我们的网站内容以横纵两条线为大方向，以小模块的形式展示细致。横向为宏观展现，其中包括志愿行动、政策支持、水质问题、美好未来、环保百科以及绿色桂电等，全方面的向人们展示我们的地球血液――水遇到的问题以及各方的救水活动，并产生一定的号召性。而纵向以小模块的细微展示形式出现，对每一个大模块大方向进行详细说明。

3、网页设计

3.1 首页设计

我们的网站属于环保型的网站，侧重点在水，采用了蓝色做为网站的总体色，使用浅灰色作为背景色，偶尔穿插多姿多彩的图片和白色背景使得网站的层次非常分明，视觉上感觉很舒适。在首页中简要的介绍了本次网站设计的主题“水对于地球来说，就像血液对于地球上的生物一样”，我们应该保护水资源、爱护水资源。最后推荐了一些关于环保方面的文章以及一些在环保方面做的比较出色的人物。

3.2 栏目设计

3.2.1 志愿行动

该模块展示的是志愿者们在用自己的行动来智力保护我们的水资源。他们自行组织，个人或者团队一起，一直默默的为我们的绿水行动，不求回报。他们用行动带动更多的人一起加入保护水的活动中来。

没有钢盔，没有军装。穿的是红马夹，戴的是绿袖章。小红旗，在手中指挥。志愿卡，在胸前闪耀。虽巳年近花甲，两鬓巳添“银霜”。但，精神依然充沛，脸上彰显“祥和”。

 

图3.2 志愿行动

3.2.2 政策支持

在国家大力发展工业时，忽略了废水排放对水质的危害。一片片的水质受到影响，国家当然也不能就这样任由发展坐视不管。在意识到情况的严峻后，国家立即颁布了一系列的政策，护水治水。在政策支持下，大力开展治水行动。在此模块中列举了一些关于水资源保护的法律条文，以此表明国家保护水源的决心；同时也列举了一些地方政府以实际行动和投入大量资金治理水源的行动，表明政府对水源治理的支持。

图3.3政策支持

3.2.3 水质问题

造成水体污染的原因很多,但是水污染物源于很多的人类活动，具体为：工业污水或废弃物随意处理、养殖场动物粪便污水排放、原油或石油的泄露、开矿产生的污水直接排放、生活污水及垃圾不合理处理等等。我们需要正视这些问题这些原因，保护水资源。本模块通过比较过去与现在的水源状况，突出现今水源状况不容乐观，同时表明一些致使水源污染的的原因，以此来号召大家参与环保，注意水资源的保护和节约用水。

图3.4水质问题

3.2.4 美好未来

古桑干河，银波泛泛，晚霞蒙蒙。彩虹辉映着湛蓝的晴空，阵阵凉风吹来，美丽的嘉陵江两岸，风光更加动人。当凉风习习低拂过水面的时候，水上顿时会出现一条瞬间即逝的狭长的银色薄箔。

水是生命之源，缺少这这源泉 ，生命就失去了意义，而今，我们正努力去保护我们身边的水源，只要人人都能狗为我们的环境贡献一份力量，努力维护好我们公共的家，我相信，我们生活的未来是一片绿意盎然，没有环境的破环，没有污染，没有垃圾，一切都是那么的美好。

图3.5美好未来

3.2.5 环保百科

古有大禹治水，现有治水环保组织，可见，治水是长久以来不变的主题。地球的血液，人类的生命源。小到节约一滴水，大到不乱排工业污水，所有的行动都能保护我们的水资源。不以善小而不为。设计本模块的目的在于介绍一些生活中的保护环境、保护水资源、节约用水的知识。通过图文结合的形式说明在生活中我们可以通过一水多用的形式节约用水、拒绝使用一次性物品以节约资源、支持绿色食品以减少化学物品的使用减少环境污染。最后呼吁大家积极参与到环保事业中去。

 

图3.6环保百科

4、技术难点及创新点

4.1 技术难点

1、颜色的搭配，网站的布局及Flash和图片的制作对于一个没有人学过设计的小组来说是很困难的。

2、使用Jquery技术与flash和图片配合制作出了动画的效果，在稳定性上更高。

3、使用Java EE强大的后台技术（可以编写java代码），配合HTML的css样式，和falsh编写出了后台强大，界面美观的网站。

4、网站实现了中英文两种版本。

4.2 创新点

1、用Jquery来实现动画效果，适应性和稳定性都比别的强，不必像flash那样要求用户装插件才能使用。

2、使用Ajax和Jquery与后台的交换技术，使网站的服务器负担减少。

3、水是地球最重要的东西，从当前水质污染非常严重的情况出发，以水为中心点延伸出环保的重要性，呼吁爱护环境。
